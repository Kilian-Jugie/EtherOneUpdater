Description détaillée

Manager pour donnés JSON V1.0.

Permet de créer, modifier des fichiers data en utilisant du json. Cette class n'est utilisble qu'avec Qt, une version STL sera probablement faite toujours avec le parser actuel, une version templated peut aussi être envisagée avec un allocator custom. Cette class reste en développement avec par exemple remToArray qui n'est actuellement pas disponible.
